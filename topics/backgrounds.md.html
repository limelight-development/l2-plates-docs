Replacing the Background

If you want non-default license plate backgrounds, you'll need to know how to use FastDL or the Workshop.

Create your License Plates

These need to be pngs, with the size 512x256. Draw in your details / top bar / whatever else.

Upload the Content

Upload your content to either the workshop or FastDL, with the path materials/ll_plates//.png

Change your Config

Change Config.Version to be whatever your was. For example, if I called mine ll_plates/internet/default.png, I would have PLATE.Config.Version = "2"

Then, follow the instructions in seasonal-plates.md.html to set the actual plate to whatever you want.
